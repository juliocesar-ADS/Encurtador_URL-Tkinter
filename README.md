🔗 Encurtador de URL com Python + Tkinter

Meu primeiro projeto com interface gráfica utilizando Python e Tkinter 🚀

Este programa permite que o usuário insira uma URL, gere um link encurtado automaticamente e copie o resultado para a área de transferência.

 Funcionalidades

✅ Interface gráfica com Tkinter
✅ Encurtamento de URLs
✅ Botão para copiar o link gerado
✅ Tratamento básico de erro
✅ Organização do código em múltiplos arquivos

🛠️ Tecnologias utilizadas
Python 3
Tkinter
pyshorteners

▶️ Como executar o projeto
1. Clone o repositório
git clone https://github.com/seu-usuario/encurtador-url.git
2. Instale a biblioteca necessária
pip install pyshorteners
3. Execute o programa
python main.py
