🔗 Encurtador de URL com Python (Tkinter)

Aplicação simples com interface gráfica feita em Python para encurtar URLs utilizando a biblioteca pyshorteners.

 Funcionalidades
Inserir uma URL
Gerar link encurtado automaticamente
Interface simples com Tkinter
Tratamento básico de erro (campo vazio)
 Tecnologias utilizadas
Python 3
Tkinter (interface gráfica)
pyshorteners (encurtamento de URL)

▶ Como executar o projeto
1. Clone o repositório
git clone https://github.com/seu-usuario/encurtador-url.git
cd encurtador-url
2. Instale as dependências
pip install pyshorteners
3. Execute o programa
python main.py
